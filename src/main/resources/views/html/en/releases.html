MultiBit Classic Releases

MultiBit Classic is:

  • free and open source under the MIT licence
  • designed to work directly with any Bitcoin full node (no servers)
  • not related to the Bitcoin Classic full node project
  • a legacy product so you should upgrade to MultiBit HD

The security model of MultiBit Classic is weaker than MultiBit HD so only use it if you have existing private keys that cannot be migrated.

MultiBit Classic is in maintenance mode. This means that only essential security and network compatibility updates will be made to the codebase.
