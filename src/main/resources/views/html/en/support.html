Fast fixes

Sometimes a transaction seems to get "stuck" leading to concern that it hasn't been sent or received. This is almost always fixed by using the "Reset Blockchain" option from the Tools menu.

Steps to take if you need help

There are various support resources online if you need help using MultiBit:

  1. Read through the MultiBit help if you have not done so already. In MultiBit choose the menu option "Help | Help Contents" or read the release documentation for your version.

  2. Check the support notes in the MultiBit forum. Topics include:

    • I opened MultiBit and my wallet(s) were missing.

    • I sent some bitcoin TO MultiBit but they never arrived.

    • I sent some bitcoin FROM MultiBit but they are stuck.

  3. A quick search on Bitcoin StackExchange can often be fruitful.

  4. If that does not resolve your issue, see if it is already raised in the MultiBit Issue reporting center. There may be a workaround.

  5. If you discover a new bug in MultiBit please report it - it makes MultiBit better for everyone. To report a bug please do the following

    1. Copy the following text: 
        Bug report
  Operating System:
  MultiBit Version:
  Steps to replicate the problem:
    2. Visit the MultiBit Issue reporting center
    3. Click on the "New Issue" button.
    4. Paste the template text and fill it in for your system. The more detail you can give the quicker we can address the problem. Screenshots are often very useful.

Due to the number of downloads of MultiBit now gets we cannot answer every support query personally. We do see these issues and make acknowledging them a priority.
