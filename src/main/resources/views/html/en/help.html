Quick fixes

Here are some quick fixes to common issues:

  • Transactions are missing
  • Lost wallet
  • Lost or forgotten password
  • Receiving doesn't work
  • Sending doesn't work
  • I'm seeing an error code

None of those helped. What now ?

Check that you're using the latest version of MultiBit (downloadVersion) using the "About" menu in MultiBit.

If you need to upgrade, do the following:

  1. Copy your existing MultiBit wallets somewhere under your exclusive control (e.g. USB stick, not a cloud backup provider)
  2. Download and install the latest version for your computer (use the links provided in the side panel).
  3. Start the upgraded version and check if the problem is still happening

I'm using downloadVersion and it's still happening. What now ?

There are lots of support resources online to help you use MultiBit. Just a little bit of searching in Google will often resolve the problem.

  1. Read through the MultiBit help (if you have not done so already).
  2. Look through the Bitcoin Stack Exchange site.
  3. Search previous MultiBit issues because someone else has almost certainly been in the same situation.

I'm the first person to have this problem! What now ?

OK, you should file an Issue. Use the approach below to get the fastest response from the support team:

  1. Copy the following text: 
      Bug report
  Operating System:
  MultiBit Version:
  Steps to replicate the problem:
  2. Re-visit the MultiBit Issue reporting center
  3. Click on the "New Issue" button.
  4. Paste the template text and fill it in for your computer. The more detail you can give the quicker we can address the problem. Screenshots are often very useful.

How to install

Windows - How to install MultiBit

Download the Windows installer.

To run the installer, double click it and follow the instructions.

Once installed, you can start MultiBit with the 'MultiBit' shortcut in your 'All Programs | MultiBit' menu or with your desktop 'MultiBit' shortcut.

Note: If you are on Windows 8 and the installer gives the error "This directory cannot be written to" this is because you need to run it as an administrator. Close the installer. Right click on it and choose the option "Run as administrator".

Mac - How to install MultiBit

Download the Mac OS X installer.

After you have downloaded the disk image file, double click it and a folder window opens.

In the open folder window, drag the MultiBit.app application to the Applications link. MultiBit will then be copied to your Applications folder

You can then open MultiBit by double clicking on this app in your Applications folder.


Linux - How to install MultiBit

Download the Linux / Unix installer.

Open a terminal window and make the installer executable with:

chmod +x multibit-downloadVersion-linux.jar

Run the installer with:

java -jar multibit-downloadVersion-linux.jar

Follow the instructions in the installer.

After installation, you will have a shortcut to start MultiBit in your "Applications | Other" menu.

Common gotchas

If you get an error like "No command 'java' found" or the installer window does not appear you probably need to install an up-to-date version of Java. See the section 'How to install Java on Linux' below.

If you see an error "***.desktop (No such file or directory)" this is a glitch in installing the MultiBit shortcuts. If you run the installer again with the same settings it normally fixes it.

If no MultiBit shortcut is created, you can run MultiBit manually as follows:

  1. open a terminal window and 'cd' to your installation directory
  2. type java -jar multibit-exe.jar

How to install Java on Linux

To install Java on a recent copy of Ubuntu open a terminal window and type:

sudo apt-get install openjdk-7-jdk

You will be prompted for your system password and then it will install the required packages.

If you are running an older version of Ubuntu (e.g. something like Ubuntu 10.4 LTS) you will have to use the following:

sudo apt-get install openjdk-6-jdk

For RedHat / CentOS the installation command is:

yum install java
