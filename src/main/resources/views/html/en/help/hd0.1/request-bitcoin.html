1. Request payment

To request bitcoin from someone else they need a "receiving address". MultiBit HD gives you a new receiving address each time you select "Send/Request" and then "Request".

  • On the "Request payment" screen you don't have to fill anything in. You can just copy the the address, by:
    • highlighting the address and using CTRL-C, or
    • selecting the copy icon, or
    • selecting the QR code icon, to get a QR code that a Bitcoin wallet on a mobile phone could use to send you bitcoin.
  • However, you can add more information about the bitcoin you want to receive. For instance, you could
    • enter a private note (e.g. "Payment for software development") to help you keep track of why you're receiving bitcoin,
    • specify the amount you expect (either in bitcoin or in the local currency),
    • add a label for the QR code. The QR code contains the address, the label and the amount all in a convenient format.
  • Click "Finish" to go back to the Send/Request screen.

Once you have requested a payment, it appears in the Payments screen with the status 'You requested'.

2. The other person then sends you bitcoin - it is initially 'Unconfirmed'

When the other person sends you bitcoin, you see at the top of MultiBit HD the message "+ Unconfirmed: mBTC xyz", where "xyz" is the amount of bitcoin they sent you.

Initially when a bitcoin transaction is sent it is not yet in a block and is 'unconfirmed'. You cannot spend unconfirmed transactions in MultiBit HD.

When your request has been paid and before it is confirmed the status of the transaction is 'Receiving'. If the other person only sends you part of the bitcoin you requested, the status is 'Part paid'.

3. Some time later your transaction is confirmed and you can spend it.

Bitcoin miners process all the unconfirmed transactions and put them into blocks. Once this happens your transaction is confirmed and you can spend it. This normally takes between 10 minutes and an hour. Once your transaction is confirmed in a block its status changes to 'Received'.

How can I find out more about a transaction ?

You can view many details of a particular transaction as follows:

  • Select a transaction in the Payments screen and click the 'Details' button (or double click the transaction) to open the 'Transaction wizard'.
  • In the transaction wizard, navigate to the screen titled 'Transaction details'. Click the 'View in blockchain.info' button to see your transaction in a block explorer.
  • MultiBit HD supports several block explorers. You can change which block explorer is used to view a transaction as described in Payment settings
