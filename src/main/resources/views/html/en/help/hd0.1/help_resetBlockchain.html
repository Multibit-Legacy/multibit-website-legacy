Reset blockchain and transactions

Occasionally you get transactions that never seem to confirm. If you have a transaction that has not confirmed for several hours but you know it is definitely on the blockchain (because you visited Blockchain.info or Blockexplorer.com and checked the transaction ID) then you can reset your blockchain.

This process gives MultiBit the opportunity to revisit old transactions. During this process MultiBit will check if a transaction has fully confirmed. If so, then it will appear in your transaction list.

How to reset the blockchain

  1. Verify that your computer's clock is showing the correct time.
  2. Open MultiBit and make sure the wallet you are interested in is selected in the ' Wallets' side panel.
  3. Go the 'Tools |  Reset Blockchain and Transactions' menu option.
  4. This brings up a new screen. To reset the selected wallet click the ' Reset blockchain and transactions' button.
  5. MultiBit will start reloading the blockchain. This can take some time. Just let it run and download the blockchain. When it is finished, your wallet should reflect what is on the blockchain.
