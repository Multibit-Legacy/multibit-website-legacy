Labs

To try out any experimental features of MultiBit HD select "Labs" from the Preferences screen

  1. From the "Preferences" screen select "Labs".
  2. In the "Labs" screen select any experimental features you want to switch on and then "Apply".
  3. You can always change your preference by returning to the "Labs" screen.

Tor is no longer supported in MultiBit HD

We have dropped Tor support from MultiBit HD 0.1.3 onwards. Unfortunately this functionality has proven too unreliable for production code. If and when this becomes more solid then we will add this functionality back into MultiBit HD.
