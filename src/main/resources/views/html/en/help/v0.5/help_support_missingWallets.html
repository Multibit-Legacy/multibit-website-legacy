I opened MultiBit Classic and my wallets were missing

What are the symptoms ?

You open MultiBit Classic and instead of seeing your wallet(s) in the Wallet list, you don't see anything. The wallet(s) have probably just been closed.

What can you do when this happens ?

Relax - you are very likely to still have your wallet(s) on your disk. Do the following:

  1. Review file locations which describes where MultiBit Classic stores the default wallet (multibit.wallet) for your computer.
  2. Have a look in that directory. You can open any wallet there using the 'File |  Open Wallet' menu option.
  3. Use your computer's search tool to find ".wallet" and include hidden directories - you should see the automatically generated backups.
  4. If you created a new wallet and stored it in another directory then you will have to look there.
  5. Check any external/network drives you may have chosen when saving the wallet

I think the files were deleted

If you've checked the file locations you should consider using file recovery software for your computer. We cannot directly recommend a particular vendor since we don't have the resources to independently verify all offerings and their latest updates. A Google search for "file recovery software" should yield popular solutions.

During the recovery process you are looking for files ending in ".wallet" or ".key" (if you've done a private key export).

Note that MultiBit Classic will not delete your wallets. If you suspect foul play you should run up to date anti-virus software to ensure you have not been infected with malware.
