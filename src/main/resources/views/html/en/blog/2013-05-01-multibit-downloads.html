MultiBit downloads for April 2013

TL;DR There were an estimated 73,200+ downloads of MultiBit in April 2013.


Estimating the number of downloads of MultiBit for April 2013 is not obvious so I thought I'd write up the methodology I have used.

The source data is here.

Normally to estimate the number of downloads I take the total bandwidth used by all the installers, DMG files, exes and jars and divide by the largest installer, which is the windows one. This should be a conservative estimate. I don't use the click counts as people use download managers and can start and stop the downloads. These bump up the click counts.

In April the size of the installers was reduced dramatically for the 0.5.9x code - to about 10MB from about 30MB.

So:

  1. Total GB downloaded on installers etc in April : 1847.81 GB.

  2. This comprised: 129.88 GB of version 0.5.9 code and 1717.93 GB of other versions.

  3. Max size of 0.5.9 code = 10.6 MB => there were about 129.88 * 1024 / 10.6 downloads of 0.5.9 code = 12546 downloads.

  4. Max size of other code = 29 MB => there were about (1847.81 - 129.88) * 1024 / 29 downloads of other versions = 60,660 downloads.

  5. Thus the total MultiBit downloads for April 2013 = 12546 + 60,660 = 73,200+ downloads.

For reference the earlier months' download figures were: Jan : 4600, Feb : 8,800, Mar : 26,000
